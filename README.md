# Помощник переименования (для Залифа)

Небольшая и удобная программа для переименования сразу многих файлов в
Windows 10. Она приветствует по имени, показывает понятный предпросмотр
"до и после" и ничего не меняет, пока не нажата зелёная кнопка. Любое
переименование можно отменить.

## Что умеет программа

- Дать всем файлам одно новое имя (например song101, song102 -> Track_1, Track_2)
- Найти и заменить текст в именах файлов
- Добавить текст в начало или конец имени
- Пронумеровать файлы: номер в начале или в конце, свой разделитель
  (например _ ), с какого числа начать и ведущие нули по желанию
- Сменить регистр букв: строчные, ПРОПИСНЫЕ или Первая Буква Каждого Слова
- Живой предпросмотр каждого "старое имя -> новое имя" перед изменением
- Подсветка красным, если у двух файлов получится одинаковое имя
- Кнопка "Отменить последнее переименование" в одно нажатие

Расширение файла (.jpg, .docx и т.д.) всегда сохраняется, поэтому файлы
продолжают открываться как раньше.

## Самый простой способ запустить (два варианта)

### Вариант А: собрать один файл .exe (лучше всего для отца)

На любом компьютере с Windows, где есть Python:

1. Дважды щёлкните по `Собрать exe.bat`.
2. Дождитесь окончания сборки.
3. Откройте появившуюся папку `dist` и скопируйте
   `Помощник переименования.exe` на компьютер отца.
4. Он просто дважды щёлкает по `Помощник переименования.exe`.
   Python на его компьютере не нужен.

### Вариант Б: запустить скрипт напрямую

1. Установите Python с сайта https://www.python.org/downloads/
   При установке поставьте галочку **"Add python.exe to PATH"**.
2. Дважды щёлкните по `Запустить помощник.bat`.

Tkinter (часть, которая рисует окно) входит в стандартный установщик
Python для Windows, поэтому больше ничего ставить не нужно.

## Как пользоваться

1. **Шаг 1** - нажмите **Обзор** и выберите папку с файлами.
2. **Шаг 2** - укажите, что изменить (например, заменить "IMG" на "Отпуск").
3. **Шаг 3** - проверьте список предпросмотра. Зелёный цвет означает, что
   имя изменится.
4. Нажмите **Переименовать файлы**, подтвердите - и готово.
5. Если что-то не так, нажмите **Отменить последнее переименование**.

## Примечания

- Переименовываются только файлы. Папки внутри выбранной папки не трогаются.
- Скрытые системные файлы (имена, начинающиеся с точки) пропускаются.
- Программа работает только с выбранной папкой, без вложенных папок. Так
  безопаснее и понятнее.
